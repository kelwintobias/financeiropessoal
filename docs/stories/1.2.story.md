# Story 1.2: Orçamento Mensal e Alertas

## Status

Ready for Review

## Story

**As a** usuário do Kekel Finance,
**I want** definir um limite de gasto mensal por categoria e receber alertas quando me aproximar ou ultrapassar esse limite,
**so that** eu tenha controle proativo dos meus gastos e evite estourar o orçamento.

## Acceptance Criteria

1. Tipo `Budget { id, categoryId, month (formato 'YYYY-MM'), limitAmount }` adicionado ao store Zustand com persistência
2. Tela de configuração de orçamentos: listar todas as categorias com campo de limite em R$ editável inline ou via formulário
3. Indicador de uso por categoria exibido na tela de orçamentos e na listagem de gastos: `"Alimentação: R$ 320,00 / R$ 500,00 (64%)"`
4. Barra de progresso colorida por categoria: verde quando < 80% do limite, amarela entre 80–100%, vermelha ao ultrapassar
5. Alerta visual (banner fixo no topo ou toast) exibido automaticamente ao registrar um gasto que faz uma categoria ultrapassar 100% do limite
6. Orçamento é mensal e independente — cada mês `YYYY-MM` tem seu próprio conjunto de limites
7. Categoria sem orçamento definido não exibe barra de progresso (campo fica vazio/0 por padrão)
8. `tsc --noEmit` e `npm run build` sem erros após implementação

### Mês Atual como String

```typescript
// Sem dependências externas — usar nativo
const currentMonth = (): string => {
  const d = new Date()
  return `${d.getFullYear()}-${String(d.getMonth() + 1).padStart(2, '0')}`
}
// ex: "2026-03"
```
