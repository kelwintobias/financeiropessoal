# Story 1.3: Metas Financeiras e Dashboard

## Status

Ready for Review

## Story

**As a** usuário do Kekel Finance,
**I want** criar metas financeiras com acompanhamento de progresso e ver um dashboard com resumo visual do meu mês,
**so that** eu tenha uma visão clara e motivadora da minha saúde financeira em um único lugar.

## Acceptance Criteria

1. Tipo `Goal { id, name, targetAmount, currentAmount, deadline?: string, status: 'active' | 'completed' | 'archived', createdAt }` adicionado ao store com persistência
2. CRUD de metas: criar (nome + valor-alvo + prazo opcional), editar, arquivar, concluir (status → 'completed')
3. Registrar aporte em uma meta: adicionar valor ao `currentAmount` (não pode exceder `targetAmount`)
4. Barra de progresso por meta com % atingido e valor restante: `"R$ 1.500 de R$ 5.000 (30%) — Faltam R$ 3.500"`
5. Dashboard (`/` rota principal) com 3 seções obrigatórias:
   - **Resumo do mês:** total gasto no mês atual vs. soma dos orçamentos definidos (ou "Sem orçamento" se nenhum)
   - **Gastos por categoria:** gráfico de pizza ou barras usando Recharts, mostrando distribuição do mês atual
   - **Metas ativas:** lista das metas com status 'active', com barra de progresso de cada uma
6. Gráfico Recharts responsivo (`width="100%"`) e acessível (não depende exclusivamente de cor — incluir legenda com texto)
7. Bottom-nav completo e funcional: Dashboard (`/`) | Gastos (`/expenses`) | Orçamento (`/budget`) | Metas (`/goals`)
8. Tela de metas em `/goals` com lista de metas ativas e arquivadas/concluídas em seções separadas
9. App funciona 100% offline — sem nenhuma chamada de rede (`fetch`, `axios`, etc.)
10. `tsc --noEmit` e `npm run build` sem erros; layout validado em 375px e 768px

### Slice Zustand — Goal

```typescript
interface GoalSlice {
  goals: Goal[]
  addGoal: (g: Omit<Goal, 'id' | 'currentAmount' | 'status' | 'createdAt'>) => void
  updateGoal: (id: string, data: Partial<Goal>) => void
  deleteGoal: (id: string) => void
  addContribution: (goalId: string, amount: number) => void
}

// addContribution implementation:
addContribution: (goalId, amount) =>
  set(state => ({
    goals: state.goals.map(g =>
      g.id === goalId
        ? { ...g, currentAmount: Math.min(g.currentAmount + amount, g.targetAmount) }
        : g
    )
  }))
```

### Recharts — Dados para PieChart

```typescript
// Transformar expenses em dados para o gráfico
type ChartData = { name: string; value: number; color: string }

const buildChartData = (
  expenses: Expense[],
  categories: Category[],
  month: string
): ChartData[] => {
  const monthExpenses = expenses.filter(e => e.date.startsWith(month))
  const grouped = categories
    .map(cat => ({
      name: cat.name,
      value: monthExpenses
        .filter(e => e.categoryId === cat.id)
        .reduce((sum, e) => sum + e.amount, 0),
      color: cat.color,
    }))
    .filter(d => d.value > 0) // só categorias com gasto
  return grouped
}
```
